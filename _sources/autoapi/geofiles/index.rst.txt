.. py:function:: prepared_geofiles(choice: str, plot: bool = False) -> tuple[str, geopandas.GeoDataFrame, str]

   Prepared geofiles for various spatial resolutions

   Parameters
   ----------
   choice : str
       Currently supports:

           'DK Municipalities'

           'NUTS1'

           'NUTS2'

           'NUTS3'

           'Nordpool'

           'NordpoolReal'

           'BalmorelVREAreas'

           'Antbalm'

           
   plot : str
       Chooses to plot or not

   Returns
   -------
   area_names : Array of str
       The region names.
       
   areas : GeoDataFrame
       The pre-processed shapefile, including all metadata.

       splitint (int, optional): _description_. Defaults to 0.
